# 🎧 Beats Landing Page – Responsive Website

A modern, responsive **Beats headphones landing page** built using **HTML, Tailwind CSS, and Font Awesome**.
This project showcases a stylish UI design inspired by premium product websites with clean layout, responsive sections, and smooth user experience.

---

## 🚀 Features

* 🎯 Clean and modern UI design
* 📱 Fully responsive (Mobile, Tablet, Desktop)
* 🎨 Tailwind CSS utility-based styling
* 🔤 Google Fonts integration
* 🎵 Product showcase section
* 🛒 Call-to-action buttons (Add to Bag, Buy Now)
* 📦 Product cards with hover effects
* 🔗 Social media icons
* 📧 Subscription section

---

## 🛠️ Technologies Used

* **HTML5**
* **Tailwind CSS (CDN)**
* **Font Awesome**
* **Google Fonts**

## 📱 Responsive Design

The layout adapts to:

* ✅ Mobile devices
* ✅ Tablets
* ✅ Laptops & desktops

Tailwind responsive utilities used:

* `sm:` → small devices
* `md:` → tablets
* `lg:` → desktops
* `xl:` → large screens

---

## 🎨 Customization

### Fonts

You can modify fonts in Tailwind config:

```js
tailwind.config = {
  theme: {
    extend: {
      fontFamily: {
        poppins: ["Poppins", "sans-serif"],
      },
    },
  },
};
```

---

### Colors

Change colors easily using Tailwind classes:

```html
bg-black
bg-stone-800
hover:bg-red-700
```
